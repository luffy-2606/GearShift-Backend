# GearShift Backend

## Overview
GearShift is a backend service designed to handle authentication, user management, and database operations. It integrates Google SignUp and Supabase authentication for secure login flows and smooth communication between frontend and backend.

## Project Structure
- config/ → configuration files (auth, environment setup)  
- models/ → database models and schema definitions  
- routes/ → API routes for user and admin operations  
- server.js → entry point for backend server  
- supabase-schema.sql → SQL schema for Supabase integration  

## Setup Instructions
Clone the repository, navigate into the project folder, install dependencies, configure environment variables in .env (see .env.example for guidance), and then run the server using npm start.
